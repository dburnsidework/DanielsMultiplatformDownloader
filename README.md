Daniel's YouTube Downloader

A sleek, lightweight YouTube and media downloader built with Python, Tkinter, and yt-dlp. This tool allows you to download videos in various qualities or convert them directly to MP3 with a single click.

Features

    Simple GUI: No command line needed. Just paste and click.

    Smart Quality Selection: Choose between 480p, 720p, or 1080p.

    Audio Conversion: Automatically converts videos to high-quality MP3 (192kbps).

    Multi-Threaded: The app stays responsive while your download happens in the background.

    Portable: The standalone .exe version includes FFmpeg, so no extra installation is required.

Built With

    Python 3.x

    yt-dlp: The powerhouse behind the media extraction.

    Tkinter: For the "Aesthetic" Windows-native GUI.

    FFmpeg: Bundled for high-quality merging and audio conversion.

 How to Run (For Users)

    Download the EXE: Go to the Releases tab.

    Launch: Run YTDownloader.exe.

    Download:

        Paste your URL.

        Select your format (MP4 or MP3).

        Hit START DOWNLOAD.

        Your file will appear in your Downloads folder!

    Note: Because this is an unsigned .exe, Windows might show a "SmartScreen" warning. Click More Info -> Run Anyway.
